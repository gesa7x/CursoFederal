    Apresentação

     

    • Neste módulo, o aluno vai ter, através da linguagem de programação C, os primeiros contatos com variáveis, estruturas sequenciais (entrada, processamento e saída de dados), estruturas de decisão, estruturas de repetição e vetores unidimensionais.

       O módulo conta com 50 horas de curso e será dividido em 5 semanas. Todos as semanas contarão com pelo menos uma atividade que farão parte da nota final do módulo.

      A nota será composta desta forma:

      As atividades das semana 1, 2, 3, 4 e 5 valerão 50% da nota final do módulo. Estas atividades serão compostas por exercícios de múltipla escolha.

      Os outros 50% da nota será dado a partir de um projeto que pediremos para todos realizarem. O projeto trata-se de um programa que os alunos deverão desenvolver e enviar para que nós possamos corrigir. O mesmo estará disponível a partir da terceira semana do curso.

      Bom estudo a todos!


    Semana 1

     

    • Lógica de Programação - Linguagem C 


      Pessoal, não deixem de ouvir este áudio. Nele tem informações importantes sobre este módulo!

      CLIQUE AQUI PARA OUVIR

      Vamos iniciar com a instalação da ferramenta DevC++.  Caso prefira outra ferramenta para trabalhar, fiquem à vontade para instalar a ferramenta de sua preferência.

      INSTALAÇÃO DEVC++

      Faça o download do compilador nos seguintes links, dependendo do seu sistema operacional:

      • Windows 8 e 10: http://sourceforge.net/projects/orwelldevcpp/files/latest/download
      • Demais Windows: http://sourceforge.net/projects/dev-cpp/files/Binaries/Dev-C%2B%2B%204.9.9.2/devcpp-4.9.9.2_setup.exe/download?use_mirror=ufpr

      Em caso de dúvidas, segue tutorial de instalação: http://linguagemc.com.br/tutorial-de-instalacao-do-dev-c/

       

      CRIANDO UM PROJETO NO DEVC++

      A seguinte vídeoaula irá lhe conduzir a fazer seu primeiro programa no DevC++. Siga atentamente a aula e pause se necessário para conseguir acompanhar passo a passo.

      Nesta aula iremos fazer o primeiro programa e também aprender ENTRADA e SAÍDA DE DADOS em Linguagem C.

      Pessoal, não deixem de desenvolver alguns programas com base nos nossos vídeos. Modifiquem os códigos e vejam o que acontece para ir acostumando com as mensagens de erro. Treinem o máximo que puderem. Este é o caminho!

      Agora que vocês entenderam como interagir com os usuário, vamos aprender a fazer algumas contas.

      Mais informações sobre este tema poderão ser vistos no capítulo 5 da apostila disponível para download no final do conteúdo desta semana.

      Ainda ficou com algumas dúvidas? Veja no link abaixo uma outra videoaula. O conteúdo é praticamente o mesmo, mas com uma outra forma de explicar. É sempre bom ver algo a mais.

      Se alguma coisa da semana anterior não ficou clara, sugiro que reforço o conhecimento, pois o nosso conteúdo é cumulativo, ou seja, o que aprendemos na semana 1, usaremos até o final deste curso. 

      Vamos iniciar com mais uma videoaula! Veremos estrutura de decisão.


      Nos dois próximos vídeos, eremos estrutura de decisão, mas agora com condições compostas. 




      Além desses vídeos, sempre costumo procurar no Youtube outros vídeos que falem deste tema. Isso permite que você vejam diferentes explicações sobre o mesmo assunto, enriquecendo o conhecimento.

    Semana 3

     

    • Loops (laços de repetição)


      Vamos iniciar com mais uma videoaula! Veremos estrutura de repetição. Primeiramente vamos entender o funcionamento do FOR. As seguintes videoaulas abordam, respectivamente: funcionamento do for e sintaxe; exercícios para fixação.



      Em paralelo, vocês devem estudar o capítulo 7 da apostila, que contém o conteúdo abordado nesta semana. No final deste módulo está também disponível para download uma lista de exercícios extras.

      Para implementar laços de repetição, além do FOR, existe também o comando WHILE. Para entender este outro comando, assistam as duas videoaulas a seguir.


      Além do comando FOR e WHILE, existe o DO WHILE. Este último é menos utilizado em relação ao FOR e WHILE, mas é importante que todos conheçam esta opção de comando.


      Como de costume, deixo abaixo uma videoaula desenvolvida por outro professor. A mesma também fala sobre FOR, WHILE e DO WHILE.

    Semana 4

     

    • Olá pessoal. 

      Nesta quarta semana do módulo 2, iremos trabalhar com o conceito de vetores. 

      Antes de iniciarmos as videoaulas com exercícios em linguagem C, sugiro que entendamos a ideia de como funcionam os vetores e para que eles servem. Notem que a videoaula abaixo utiliza o visualg e não a linguagem C. Cuidado para não se confundirem com isso. Fiquem atentos aos conceitos e não aos comandos específicos!


      O conceito de funcionamento dos vetores é praticamente o mesmo em qualquer linguagem. Vamos então aplicar este conhecimento obtido na linguagem C. Vejam para isso as duas videoaulas abaixo.


      Além dos vetores, existe o conceito de matrizes também. Não cobraremos este conhecimento neste curso, mas é muito importante que nós programadores estejamos sempre buscando aumentar os conhecimentos. Para aqueles que tem interesse, segue abaixo mais uma videoaula. Esta apresenta o funcionamento das matrizes em linguagem C.


      Como de costume, além do vídeo, sugiro a leitura do capítulo 8 da apostila.

    Projeto Final

    Pessoal.

    Este é o projeto final do Módulo 2. Notem que vocês deverão desenvolver o projeto sugerido e enviar o arquivo CPP, ou C ou outra extensão que seja possível termos acesso ao código fonte do projeto através da plataforma (veja no final desta página o link para a atividade). Não enviem o arquivo DEV ou EXE. Com estes tipos de arquivos, nós não temos acesso ao código fonte!

    Este projeto deverá ser entregue até o dia 20 de março de 2017, as 23 horas e 55 minutos. Fiquem atentos à esta data, pois não poderemos de forma alguma modifica-la.

    Notem que até este dia, teremos completado 4 semanas do curso. O conteúdo da quinta semana não será cobrado no projeto. 

    Na quinta semana, uma atividade com exercícios de múltipla escolha com data de vencimento no dia 27 de março será passado para vocês. Para mais detalhes, acessem o calendário do curso clicando aqui.

    Lembro que as atividades das semana 1, 2, 3, 4 e 5 valerão 50% da nota final do módulo. Estas atividades serão compostas por exercícios de múltipla escolha. Os outros 50% da nota será dado a partir deste projeto. 

    É muito importante que todos realizem todas as atividades para que a nota 6 seja atingida sem nenhum transtorno. Em caso de dúvidas, não desista. Procure a gente e faremos o possível para auxilia-los.

    Para auxiliar vocês no desenvolvimento do projeto, criei esta videoaula revisando alguns comandos que serão necessários.


    Vejam abaixo a atividade proposta. A submissão dos trabalhos pode começar a ser feita.
